Simplex Method - Steps to Solve

The Simplex Method is an iterative algorithm used to solve linear programming problems with more than two variables. It systematically moves from one feasible solution to another, in search of the optimal solution, using a tabular representation called the simplex tableau.

Steps to Solve a Linear Programming Problem Using the Simplex Method

  1. Formulate the Linear Programming Problem:
    • Decision Variables: Define the variables to be determined.
    • Objective Function: Establish the function to maximize or minimize.
    • Constraints: List all the limitations as linear inequalities.
  2. Convert Inequalities to Equations by Adding Slack Variables:
    • Add slack variables to convert â‰¤ constraints into equalities.
    • Slack variables represent unused resources and are non-negative.
  3. Set Up the Initial Simplex Tableau:
    • Create a tableau including coefficients of decision and slack variables.
    • The tableau consists of rows for constraints and one row for the objective function.
  4. Identify the Pivot Element:
    • Entering Variable: The variable with the most negative coefficient in the objective function row.
    • Leaving Variable: Determined by the minimum ratio test (RHS divided by pivot column entries).
  5. Perform Row Operations to Form a New Tableau:
    • Pivot Row: Divide the entire pivot row by the pivot element to make it 1.
    • Other Rows: Make the pivot column entries zero in all other rows.
  6. Check for Optimality:
    • If all coefficients in the objective function row are non-negative, an optimal solution is reached.
    • Otherwise, repeat steps 4 and 5.
  7. Read the Solution from the Final Tableau:
    • The values of decision variables are in the RHS column corresponding to their basic variables.
    • The optimal value is in the RHS of the objective function row.
  8. Interpret the Results:
    • Provide the optimal values of the decision variables and the objective function, ensuring practical applicability.

Example Problem: Maximizing Z

Problem Statement:

Maximize \( Z = 3x_1 + 2x_2 \)

Subject to:

  • \( x_1 + x_2 \leq 4 \)
  • \( x_1 \leq 2 \)
  • \( x_2 \leq 3 \)
  • \( x_1, x_2 \geq 0 \)

Step-by-Step Solution

  1. Define Decision Variables and Objective Function:

    \( x_1, x_2 \) are the decision variables.

    Objective Function: Maximize \( Z = 3x_1 + 2x_2 \).

  2. Convert Inequalities to Equations with Slack Variables:
    • \( x_1 + x_2 + s_1 = 4 \)
    • \( x_1 + s_2 = 2 \)
    • \( x_2 + s_3 = 3 \)
    • Slack Variables: \( s_1, s_2, s_3 \geq 0 \)
  3. Set Up Initial Simplex Tableau:
    Basis \( x_1 \) \( x_2 \) \( s_1 \) \( s_2 \) \( s_3 \) RHS
    \( s_1 \) 1 1 1 0 0 4
    \( s_2 \) 1 0 0 1 0 2
    \( s_3 \) 0 1 0 0 1 3
    \( Z \) -3 -2 0 0 0 0
  4. First Iteration - Identify Pivot Element:
    • Entering Variable: \( x_1 \) (most negative in \( Z \) row)
    • Leaving Variable: \( s_2 \) (smallest positive ratio \( \frac{2}{1} = 2 \))
    • Pivot Element: 1 (Row 2, Column \( x_1 \))
  5. Perform Row Operations:
    • Make pivot element 1 (it's already 1).
    • Eliminate \( x_1 \) from other rows.
    • Update the tableau accordingly.

    Updated Tableau:

  6. Second Iteration - Check for Optimality:
    • Check \( Z \) row for negative indicators.
    • Since \( -2 \) (for \( x_2 \)) is still negative, proceed with another iteration.
    • Entering Variable: \( x_2 \)
    • Leaving Variable: \( s_3 \) (smallest positive ratio)
    • Pivot Element: 1 (Row 3, Column \( x_2 \))
  7. Perform Row Operations:
    • Make pivot element 1.
    • Eliminate \( x_2 \) from other rows.
    • Update the tableau accordingly.

    Updated Tableau:

  8. Check for Optimality:
    • All indicators in the \( Z \) row are non-negative.
    • The current solution is optimal.
  9. Read the Solution from the Final Tableau:
    • \( x_1 = 2 \)
    • \( x_2 = 3 \)
    • Optimal Value \( Z = 3(2) + 2(3) = 12 \)
  10. Interpret the Results:

    The optimal solution is to choose \( x_1 = 2 \) and \( x_2 = 3 \) to achieve a maximum value of \( Z = 12 \).
